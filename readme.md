# 우리의 만족은 어디서 오는가?
### 양천구를 중심으로


만족스러운 삶은 인간이라면 누구나 추구하는 것이다.

지역에 대한 만족은 거주민들에게 지역에 대한 자긍심을 심어주며,
이는 자연스럽게 도시경쟁력으로 이어진다.

그렇다면 만족은 어디서 오는 것일까? 

지역만족도에 영향을 미치는 요인을 양천구를 중점으로 알아보고, 
그 결과를 바탕으로 양천구의 개발 방향성을 제안한다.
